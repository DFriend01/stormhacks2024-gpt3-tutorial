# GPT Tutorial Notebook

The notebook is publicly available on [Google Colab](https://colab.research.google.com/drive/1eq3SBle28Pjo7-e0UZxuIxZgKmLdR1JP?usp=sharing).
Feel free to follow the instructions. There are also instructions on how to run locally below.

## Running Locally

1. Install Python. You can follow the instructions [here](https://www.python.org/downloads/).

2. You need some sort of notebook kernel to run locally. The easiest way to do this is through
VS Code. You can install VS Code [here](https://code.visualstudio.com/download).

3. Open VS Code and install the following extensions
    - [Python extension for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
    - [Pylance](https://marketplace.visualstudio.com/items?itemName=ms-python.vscode-pylance)
    - [Jupyter Extension Pack](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)

4. Reload your VS Code window: `ctrl + shift + p` > Reload Window

5. Open the notebook file and run the code!
